## Scripts

- clone repository

```zsh
git clone https://github.com/SSAF-SOUND/ssaf-sound-fe.git
```

- install dependencies

```zsh
yarn
```

- dev server

```zsh
yarn dev
```

- storybook

```zsh
yarn sb
```

- cypress test

```zsh
yarn sb
yarn cy:open
```

- jest test

```zsh
yarn test
```
